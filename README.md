 AI Personal Assistant

Overview
A simple AI-powered personal assistant that can respond to:
- Time queries
- Math calculations 
- Web search 
- Basic reminders
- Voice input in the browser

Built with Python, Flask, spaCy, pyttsx3, and WolframAlpha API.
